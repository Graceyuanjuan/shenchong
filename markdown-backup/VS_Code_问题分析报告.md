## 📋 问题类型详解

### MD022 - 标题空行问题

- **描述**: 标题前后缺少空行
- **影响**: 仅影响文档可读性
- **数量**: ~80个
- **示例**:
  ```markdown
  # 错误格式
  #### 问题描述
  这里缺少空行
  
  # 正确格式
  
  #### 问题描述
  
  这里有空行分隔
  ```

### MD032 - 列表空行问题

- **描述**: 列表前后缺少空行
- **影响**: 仅影响文档渲染效果
- **数量**: ~60个
- **示例**:
  ```markdown
  # 错误格式
  段落文字
  - 列表项1
  - 列表项2
  继续段落
  
  # 正确格式
  段落文字
  
  - 列表项1
  - 列表项2
  
  继续段落
  ```

### MD031 - 代码块空行问题

- **描述**: 代码块前后缺少空行
- **影响**: 仅影响文档格式
- **数量**: ~40个
